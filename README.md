# Fullstack React + Node Starter

This repository contains a complete **React (Vite + Tailwind)** frontend and a **Node.js (Express)** backend.
Frontend is in `/frontend` and backend in `/backend`.

## Quick start
1. Install dependencies:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```
2. Start backend (from `/backend`):
   ```bash
   npm run dev
   ```
3. Start frontend (from `/frontend`):
   ```bash
   npm run dev
   ```

Frontend runs on Vite (default http://localhost:5173). Backend runs on http://localhost:4000 by default.
